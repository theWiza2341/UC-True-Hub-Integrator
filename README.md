# True Hub Bridge

An UnderScript-compatible enhancement for the Undercards Deck Hub.

True Hub Bridge replaces the traditional Hub deck browser with a faster, filter-focused interface designed to make finding decks significantly easier.

## Features
Enhanced Deck Browser
Browse all Hub decks through a dedicated "True Hub" view.
Clean pagination system.
Faster navigation through large deck collections.
Advanced Filtering

Filter decks by:

- Soul
- Included cards
- Excluded cards
- Deck name / search text

Filters can be combined to quickly locate specific deck archetypes.

Deck Preview Support

Preview deck contents directly from the Hub without manually opening every deck page.

## UnderScript Integration

Native UnderScript support provides:

Automatic update checking
Plugin management integration
Settings menu support
Built-In Settings

Current settings include:

- Auto Open True Hub
- Automatically switches to True Hub when opening the Deck Hub.
- Enable Scroll Paging
- Allows mouse wheel navigation between pages.

## Installation Requirements

You must have:

- Tampermonkey (or another userscript manager)
- UnderScript installed
- Install True Hub Bridge

Install the latest version from:

https://raw.githubusercontent.com/theWiza2341/UC-True-Hub-Integrator/main/True-Hub-Bridge.js

After installation, refresh Undercards and open the Deck Hub.

## Usage
1. Open the Deck Hub.
2. Click Switch to True Hub.

Or enable:

Auto Open True Hub

from the UnderScript settings menu to enter True Hub automatically.

## Settings

UnderScript → Plugins → True Hub Bridge

Available options:

### Setting	Description
Auto Open True Hub	Automatically opens True Hub when entering the Deck Hub
Enable Scroll Paging	Allows page switching with the mouse wheel
